# ‚≠ê 1. What is Web 3.0?

Web 3.0 shifts from platform-controlled internet (Web 2.0) to **ownership-based internet**, where users control:

* identity
* assets
* data
* interactions via blockchain

**Core idea:**

> Programmable ownership + decentralization.

**Building blocks:**

* Blockchain
* Smart contracts
* Wallets
* Tokens/NFTs
* Decentralized storage (IPFS)

# ‚≠ê 2. Evolution of the Web (Very Simple)

### **Web 1.0 ‚Üí Read**

Static pages, no user-generated content.

### **Web 2.0 ‚Üí Read + Write**

Platforms (YouTube, Facebook), user content, but centralized control.

### **Web 3.0 ‚Üí Read + Write + Own**

Users own assets/data, dApps replace platforms, blockchain = trust layer.

# ‚≠ê 4. What Are DApps (Decentralized Applications)?

DApps run on blockchain using smart contracts.

They must be:

1. **Open-source**
2. **Decentralized** (data on blockchain)
3. **Token-powered**
4. **Have a built-in consensus mechanism**

### **Types of DApps**

* **Type I** ‚Äì use their own blockchain (e.g., Bitcoin)
* **Type II** ‚Äì built on Type I (e.g., Omni on Bitcoin)
* **Type III** ‚Äì built on Type II protocols

**Unique feature:**

> Once deployed, they run without a centralized authority.

---

# ‚≠ê 5. Bitcoin as a DApp

Bitcoin **is a Type I DApp** because:

* It is open-source
* Completely decentralized
* Uses native token (BTC)
* Uses PoW for consensus

---

# ‚≠ê 6. Why Ethereum Matters

Ethereum = blockchain for **smart contracts + dApps**.

### **Key innovations**

* EVM (Ethereum Virtual Machine)
* Smart contracts
* Tokens (ERC-20, ERC-721 NFTs)
* Large ecosystem of DApps

Ethereum allows programmable logic, unlike Bitcoin.
