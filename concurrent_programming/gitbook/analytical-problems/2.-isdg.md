```c
for (int i = 1; i < 100; i++) {
    for (int j = 1; j < 200; j++) {
        data[i][j] = data[i-1][j] + data[i-1][j-1];
    }
}
```

ISDG shows vertical arrows (i depends on i-1) but **no horizontal arrows** (j iterations are independent).

→ Parallelise the j loop:

```c
for (int i = 1; i < 100; i++) {
    #pragma omp parallel for
    for (int j = 1; j < 200; j++) {
        data[i][j] = data[i-1][j] + data[i-1][j-1];
    }
}
```
