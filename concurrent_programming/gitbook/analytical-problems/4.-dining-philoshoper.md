# 4. Dining Philoshoper

Ok — quick explanation first, then answer.

***

**The problem:**

```
5 philosophers sit around a table
5 chopsticks — one between each pair
Each philosopher:
  → thinks
  → picks up LEFT chopstick
  → picks up RIGHT chopstick
  → eats
  → puts down both chopsticks
  → repeat
```

***

**The problem — deadlock:**

```
All 5 pick up LEFT chopstick simultaneously
All 5 wait for RIGHT chopstick
Nobody can proceed → deadlock forever
```

***

**The solution — resource ordering:**

```
Philosophers 0-3 → pick up LEFT first, then RIGHT
Philosopher 4    → pick up RIGHT first, then LEFT
→ cycle broken → no deadlock
```

***

**Mapped to OpenMP tasks:**

```
Each philosopher = one task
Chopstick = omp_lock_t
Pick up = omp_set_lock
Put down = omp_unset_lock
```

***

Now the exam answer:

```c
#include <stdio.h>
#include <omp.h>
#include <unistd.h>

#define N 5   // number of philosophers and chopsticks

omp_lock_t chopstick[N];  // one lock per chopstick

void philosopher(int id) {
    int left  = id;           // left chopstick index
    int right = (id + 1) % N; // right chopstick index

    // Think
    printf("Philosopher %d is thinking\n", id);
    sleep(1);

    // Pick up chopsticks — resource ordering to prevent deadlock
    if (id < N - 1) {
        // Philosophers 0-3: pick up left first then right
        omp_set_lock(&chopstick[left]);
        printf("Philosopher %d picked up LEFT chopstick %d\n", id, left);
        omp_set_lock(&chopstick[right]);
        printf("Philosopher %d picked up RIGHT chopstick %d\n", id, right);
    } else {
        // Philosopher 4: pick up right first then left — breaks deadlock cycle
        omp_set_lock(&chopstick[right]);
        printf("Philosopher %d picked up RIGHT chopstick %d\n", id, right);
        omp_set_lock(&chopstick[left]);
        printf("Philosopher %d picked up LEFT chopstick %d\n", id, left);
    }

    // Eat
    printf("Philosopher %d is EATING\n", id);
    sleep(1);

    // Put down both chopsticks
    omp_unset_lock(&chopstick[left]);
    omp_unset_lock(&chopstick[right]);
    printf("Philosopher %d finished eating\n", id);
}

int main() {
    // Initialise all chopstick locks
    for (int i = 0; i < N; i++)
        omp_init_lock(&chopstick[i]);

    // Create one task per philosopher
    #pragma omp parallel
    {
        #pragma omp single
        {
            for (int i = 0; i < N; i++) {
                #pragma omp task firstprivate(i)
                {
                    philosopher(i);
                }
            }
        }
    }

    // Destroy locks
    for (int i = 0; i < N; i++)
        omp_destroy_lock(&chopstick[i]);

    return 0;
}
```

***

**Description for exam (write this too):**

```
- N philosophers, N chopsticks represented as omp_lock_t array
- Each philosopher is an independent task — parallel single → task
- firstprivate(i) — each task gets its own philosopher ID
- Resource ordering — philosopher 4 picks up RIGHT first
  → breaks the circular wait → prevents deadlock
- omp_set_lock → acquire chopstick (blocks if held)
- omp_unset_lock → release chopstick
```

***

**The one line examiner wants to see:**

> _"Deadlock is prevented by resource ordering — philosopher N-1 acquires chopsticks in reverse order, breaking the circular wait condition."_
